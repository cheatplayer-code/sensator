# SmartPasture Compose Android

Нативный Android Studio проект на **Kotlin + Jetpack Compose**, собранный по мотивам загруженного Figma Make / React-кода.

## Что перенесено
- экран **Карта** с интерактивной картой Жамбылской области и точками
- экран **Калькулятор HPS / PNS** с формулой и расчётом потребности в воде по составу стада
- экран **Где брать данные** с очищенной структурой без лишнего мусора
- экран **Сценарии** с рисками, экономикой и action-steps

## Что я сознательно изменил
- web-hover tooltip заменён на нормальный мобильный detail card
- большая web-таблица сценариев переведена в мобильные stacked cards
- экран `KnowledgeBase` очищен от длинных таблиц по животным и формульного спама, как ты и просил раньше
- карта сделана через **osmdroid / OpenStreetMap**, чтобы не требовать Google Maps API key

## Как открыть
1. Распакуй архив.
2. Открой папку проекта в Android Studio.
3. Дождись Gradle Sync.
4. Выбери эмулятор или телефон.
5. Нажми **Run**.

## Если Android Studio пожалуется на wrapper
В этой среде невозможно было положить `gradle-wrapper.jar`, поэтому:
- либо дай Android Studio самой создать / подтянуть wrapper при sync
- либо выполни локально команду `gradle wrapper`

После этого проект синхронизируется как обычный Android-проект.
